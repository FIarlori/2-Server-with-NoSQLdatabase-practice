## Usage

To use the API:

- `npm run start`
- Enter URL or send:
  GET request in Postman to `"http://localhost:3000/health"`
  GET request in Postman to `"http://localhost:3000/questionnaire/:id"`
  POST request in Postman to `"http://localhost:3000/questionnaire"`
